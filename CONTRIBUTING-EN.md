# How to Submit a Project

## Format Template

```
#### Creator Name (City) - [Github](), [Blog]()
* :white_check_mark: [Product Name](URL): A short description, explain what the product does as clearly as possible - [More Info]()
```

City, Github, Blog, and More Info are all optional.

## Format Template - Minimal Version (required fields only)

```
#### Creator Name
* :white_check_mark: [Product Name](URL): A short description, explain what the product does as clearly as possible
```

## How to Write a Good One-Line Description

**First define the category, then add the highlights**: Start with the shortest possible phrase that says what the product is, then mention what makes it stand out.

```
[Product Type], [Core Value or Unique Feature]
```

Here are real examples from this list for reference:

- `AI voice cloning tool, clone any voice in 3 seconds, supports 10 languages including Chinese, English, Japanese, and Korean, provides 50+ preset voice library`
- `Image compression tool, 70%+ compression rate while preserving image quality, supports PNG/JPEG/WebP batch processing, runs entirely locally with no uploads`
- `Chrome bookmark manager extension, replaces the default new tab page, supports folder card view, drag-and-drop sorting, and batch operations`
- `Markdown to Word/PDF tool, supports LaTeX formulas and Mermaid diagrams, runs entirely in the browser with no registration required`
- `AI data workbench, supports MySQL/Postgres/SQLite/Clickhouse, can replace DBeaver and Navicat`
- `Mac menu bar quick note tool, floating globally without switching screens, supports mixed text and images and one-click clipboard history access`
- `AI Agent security sandbox, run Claude Code, Codex, and other Agents in fully isolated containers with a single command, zero risk to the host machine`
- `Group meeting point recommendation tool, input multiple addresses to automatically calculate a fair midpoint, covers 350+ cities, free and no registration required`
- `Multilingual listening practice app, supports importing Podcast and YouTube links, provides bilingual subtitles and read-along exercises`
- `Minimalist expression platform, no account, no interaction, no traces, perfect for lightweight "say it and let it go" expression needs`

When there are no unique features to highlight, adding specific numbers or usage constraints (free, no registration, local processing) can also make the description more valuable.
