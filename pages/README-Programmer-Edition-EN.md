## Chinese Independent Developer Projects (Programmer Edition)

[Main page click here](https://github.com/1c7/chinese-independent-developer/)

**Difference between the Programmer Edition and the Main Edition**:
* Programmer Edition: The users are programmers who can use the command line and know things like `npm install`. The products listed are open-source blogs / CLI tools, etc.
* Main Edition: The users are not programmers. Products are websites/apps/desktop applications that must work out of the box.
