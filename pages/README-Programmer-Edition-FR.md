## Projets de Développeurs Indépendants Chinois (Édition Programmeur)

[Page principale cliquez ici](https://github.com/1c7/chinese-independent-developer/)

**Différence entre l'Édition Programmeur et l'Édition Principale** :
* Édition Programmeur : Les utilisateurs sont des programmeurs qui savent utiliser la ligne de commande et connaissent des commandes comme `npm install`. Les produits listés sont des blogs open-source / outils en ligne de commande, etc.
* Édition Principale : Les utilisateurs ne sont pas des programmeurs. Les produits sont des sites web / applications / applications de bureau qui doivent fonctionner immédiatement.
